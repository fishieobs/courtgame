## 容積的代價（chapter4.html）

取材真實司法案件改編。開場可選擇立場：

- **辯護律師篇**：扮演張律，逐一檢驗檢方證據——打掉 1500 萬的指控，卻仍要面對天平的另一端。
- **檢察官篇**：扮演嚴正雄，從偵查、約談、搜索到公訴——守住對價與容積的地基，也要承受被辯方拆掉王牌的瞬間。

兩條路線的遊戲歷程完全不同（不同證物、不同詰問、不同對手），但結局相同：**被告被判處應執行有期徒刑十七年、褫奪公權六年**。宣判程序完整呈現——主文逐罪宣讀、應執行刑、沒收、判決理由（含 1500 萬不能證明之剔除）、量刑審酌與上訴告知。

兩條路線皆零失誤通關，可解鎖雙立場結語。

## 幕間支線（穿插小遊戲）

本傳的「案件選擇」畫面，章節之外另有**幕間支線**，隨時可玩：

- **蟑螂大作戰** 🪳：張律帶女兒小寶出國渡假，飯店一開燈竟滿地蟑螂——限時點擊拍扁牠們的休閒小遊戲。
- **世足射門挑戰** ⚽：張律帶小寶飛到美國看世界盃，在球場 PK 體驗區挑戰射門——門將「不規律」移動防守、還會魚躍撲救，看準空檔點擊球門位置射門得分。

每個幕間達標與否會走到不同的劇情收尾，最佳紀錄會存在裝置裡。幕間是輕鬆向的支線，沒有罰則、不影響章節進度，可重複挑戰。

## 操作

- 點擊畫面推進對話；證言（綠字）用 ◀ ▶ 切換。
- **等一下**＝追問證言（不受罰）；**出示證物**＝指出矛盾（出錯扣 ❗，5 次歸零即失敗，可原地重試）。
- 攻略見《[逆轉律師攻略.md](逆轉律師攻略.md)》（本傳）。
