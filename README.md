## 容積的代價（chapter4.html）

取材真實司法案件改編。開場可選擇立場：

- **辯護律師篇**：扮演林辯，逐一檢驗檢方證據——打掉 1500 萬的指控，卻仍要面對天平的另一端。
- **檢察官篇**：扮演嚴正雄，從偵查、約談、搜索到公訴——守住對價與容積的地基，也要承受被辯方拆掉王牌的瞬間。

兩條路線的遊戲歷程完全不同（不同證物、不同詰問、不同對手），但結局相同：**被告被判處應執行有期徒刑十七年、褫奪公權六年**。宣判程序完整呈現——主文逐罪宣讀、應執行刑、沒收、判決理由（含 1500 萬不能證明之剔除）、量刑審酌與上訴告知。

兩條路線皆零失誤通關，可解鎖雙立場結語。

## 操作

- 點擊畫面推進對話；證言（綠字）用 ◀ ▶ 切換。
- **等一下**＝追問證言（不受罰）；**出示證物**＝指出矛盾（出錯扣 ❗，5 次歸零即失敗，可原地重試）。
